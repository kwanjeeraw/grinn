getGrinnDb - get Grinn database location

Description

get Grinn database location of the current working envornment. By default Grinn database lacation is http://grinn.genomecenter.ucdavis.edu:7474/db/data/cypher which contains only human database.

Usage

getGrinnDb()

Value

url of Grinn database location.
